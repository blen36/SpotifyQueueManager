# Spotify Queue Manager
Описание проекта
Spotify Queue Manager — это веб-приложение на Django, которое позволяет создавать совместные музыкальные очереди для групповых поездок или мероприятий. Система позволяет одному пользователю (хосту) создать комнату с доступом к Spotify, а гостям — добавлять треки в очередь воспроизведения без необходимости иметь аккаунт Spotify Premium.

Ключевые возможности
Для хоста (организатора):
Создание приватных комнат для управления музыкальной очередью

Авторизация через Spotify OAuth для доступа к API

Выбор устройства воспроизведения (например, автомобильная акустическая система)

Управление очередью треков с возможностью просмотра добавленных гостями композиций

Для гостей:
Подключение к комнате через QR-код или код доступа

Поиск треков в каталоге Spotify

Добавление найденных треков в общую очередь

Просмотр текущей очереди воспроизведения

Технологический стек
Бэкенд: Django, Python

Фронтенд: HTML, CSS, Bootstrap

API интеграция: Spotify Web API

Аутентификация: Spotify OAuth 2.0

База данных: SQL Server (для разработки)

Как это работает
Хост создает комнату и авторизуется через Spotify

Гости присоединяются к комнате через QR-код

Гости ищут треки через интегрированный поиск Spotify

Система автоматически добавляет выбранные треки в очередь устройства хоста

Музыка воспроизводится напрямую через Spotify на устройстве хоста

Особенности архитектуры
Гостям не требуется аккаунт Spotify или Premium-подписка

Музыка воспроизводится исключительно через Spotify

Приложение не хранит и не передает аудиофайлы

Весь аудиопоток обрабатывается на стороне Spotify

Система работает только как управляющий интерфейс для очереди

Требования к аккаунтам
Хост: Требуется Spotify Premium и учетная запись разработчика

Гости: Не требуется Spotify аккаунт или подписка

Разработчик: Бесплатный аккаунт Spotify Developer

Используемые разрешения Spotify API
user-modify-playback-state — для управления очередью воспроизведения

user-read-playback-state — для чтения состояния воспроизведения

user-read-currently-playing — для отслеживания текущего трека

Страницы приложения
Главная страница — создание и подключение к комнатам

Страница авторизации — вход через Spotify для хоста

Страница комнаты — управление очередью для хоста

Страница поиска — поиск треков в Spotify

Страница очереди — просмотр текущей очереди воспроизведения

Преимущества системы
Простота использования: Гости добавляют треки в пару кликов

Безопасность: Музыка воспроизводится через официальный Spotify

Совместимость: Работает с любым устройством, поддерживающим Spotify

Масштабируемость: Подходит для небольших компаний и крупных мероприятий

Идеально подходит для автомобильных поездок, домашних вечеринок, офисов и любых мероприятий, где нужно коллективное управление музыкальным сопровождением.
